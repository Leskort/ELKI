# Фоновые видео

Поместите ваше фоновое видео в эту папку с именем `horror-background.mp4` или `horror-background.webm`.

## Рекомендации:

- Формат: MP4 (H.264) или WebM
- Разрешение: 1920x1080 или выше
- Длительность: 10-30 секунд (будет зациклено)
- Размер файла: старайтесь минимизировать (используйте сжатие)
- Стиль: темное, атмосферное, подходящее для хоррор-темы

## Примеры тем для видео:

- Тёмный лес с ветвями на ветру
- Мерцающие огни в темноте
- Туман или дым
- Абстрактные тёмные текстуры
- Зимний лес в сумерках

## Альтернатива:

Если у вас нет видео, можно использовать YouTube видео через iframe или найти бесплатные видео на:
- Pexels Videos
- Pixabay Videos
- Unsplash Videos
